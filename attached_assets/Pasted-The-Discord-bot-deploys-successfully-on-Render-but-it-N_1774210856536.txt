## 1. Verify `client.login()` is actually reached

```js

```

## 2. Fix READY EVENT (VERY LIKELY ISSUE)

Check your event file:
`/events/clientReady.js`

Ensure it is EXACTLY:

```js
module.exports = {
  name: "ready",
  once: true,
  execute(client) {
    console.log(`[READY] Logged in as ${client.user.tag}`);
  }
};
```

### Common mistake:

* Using `"clientReady"` instead of `"ready"`
  → This will NEVER fire

---

## 3. Fix Event Loader

Ensure event loader uses:

```js
client.once(event.name, (...args) => event.execute(...args));
```

IF `once: true`

Otherwise:

```js
client.on(event.name, (...args) => event.execute(...args));
```
