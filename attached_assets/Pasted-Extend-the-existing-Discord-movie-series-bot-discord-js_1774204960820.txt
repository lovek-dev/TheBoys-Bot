## 1. 🗳️ Majority Control System

### Commands:

* `/pause`
* `/resume`
* `/skip`

### Behavior:

* When a user runs a command:

  * Bot creates a **vote system**
  * Store voters per session:

    ```
    guildId
    sessionId
    action (pause/resume/skip)
    voters[]
    ```

* Threshold:

  * Minimum: 50% of active users in session
  * Active users = users in voice channel OR marked “ready”

### Output:

```
Vote to pause: 2/5
```

* When threshold reached:

```
✅ Vote passed. Pausing movie.
```

### Requirements:

* Prevent duplicate votes
* Auto-expire vote after 60 seconds
* Only allow users in session to vote
